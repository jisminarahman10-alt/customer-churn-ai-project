# Customer Churn Prediction AI Project

## Overview
This project builds a machine learning model to predict customer churn.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask

## Steps
1. Data preprocessing
2. Feature encoding
3. Model training
4. Model evaluation
5. Model saved as .pkl
6. API deployment using Flask

## API Endpoint
POST /predict

Returns prediction whether a customer will churn or not.
